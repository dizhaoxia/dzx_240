## 1. 产品概述

在线脑筋急转弯游戏是一款轻量级的趣味答题小游戏，用户通过随机抽取的脑筋急转弯题目进行限时作答，挑战自己的思维灵活度，享受闯关乐趣。

- 目标用户：喜欢休闲益智类游戏的用户，适合碎片化时间游玩
- 核心价值：通过趣味问答激发思维，提供轻松有趣的闯关体验

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：游戏标题、开始按钮、规则说明
2. **答题界面**：题目展示、倒计时、选项/输入、得分、关卡信息
3. **关卡过渡界面**：过关提示、进入下一关
4. **游戏结束界面**：最终得分、答对题数、重新开始

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | Hero区域 | 展示游戏名称、趣味动画、开始游戏按钮 |
| 游戏主界面 | 规则说明 | 简要展示游戏规则：每题15秒、每关5题、答对+10分 |
| 答题界面 | 题目展示 | 显示当前脑筋急转弯题目，高亮关卡和题号 |
| 答题界面 | 倒计时 | 15秒环形倒计时，时间不足时变红闪烁 |
| 答题界面 | 选项区 | 4个选项按钮，点击后即时判定对错 |
| 答题界面 | 状态栏 | 显示当前得分、关卡、题号进度 |
| 关卡过渡 | 过关动画 | 显示"恭喜过关"提示，进入下一关按钮 |
| 游戏结束 | 结算面板 | 显示总得分、答对题数、评级、重新开始按钮 |

## 3. 核心流程

用户打开页面 → 看到游戏主界面 → 点击"开始游戏" → 进入第1关第1题 → 15秒倒计时开始 → 用户选择答案 → 判定对错并反馈 → 进入下一题 → 5题答完 → 显示过关提示 → 进入下一关 → 所有题目答完或用户主动结束 → 显示游戏结果

    "K" --> "E"

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深邃星空紫 (#1a1a2e) + 霓虹青色 (#00f5d4) + 活力橙 (#ff6b35)
- 按钮：圆角胶囊按钮，带霓虹发光效果和hover缩放动画
- 字体：标题使用趣味粗体，正文使用清晰易读字体
- 布局：居中卡片式布局，深色背景配合星空粒子效果
- 图标风格：使用lucide-react图标库，霓虹风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 游戏主界面 | Hero区域 | 星空背景、大标题动画、脉冲发光开始按钮 |
| 游戏主界面 | 规则说明 | 半透明卡片、图标+文字组合 |
| 答题界面 | 题目展示 | 发光边框卡片、大号题号、题目文字 |
| 答题界面 | 倒计时 | 环形进度条、数字倒计时、低于5秒闪烁 |
| 答题界面 | 选项区 | 4个选项按钮，正确绿色/错误红色反馈 |
| 答题界面 | 状态栏 | 顶部固定、得分、关卡进度条 |
| 关卡过渡 | 过关动画 | 星星粒子爆炸、文字渐入动画 |
| 游戏结束 | 结算面板 | 居中卡片、大号得分、星级评级、重玩按钮 |

### 4.3 响应式设计

- 桌面端优先（1920x1080为基准）
- 平板适配（768px断点调整卡片宽度和字号）
- 移动端适配（375px断点，选项纵向排列，字号缩小）
